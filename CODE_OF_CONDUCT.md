# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## In short

- Be respectful. Disagree on technical content, not on people.
- No harassment, no discrimination, no personal attacks.
- Assume good intent. Ask clarifying questions before escalating.
- Keep discussions on-topic: Odoo implementation, configuration, and migration.

## Reporting

Report unacceptable behaviour to **conduct@roekish.com**. Reports are handled confidentially by the Roekish leadership.

## Scope

This Code applies to all project spaces — issues, PRs, discussions, and any communication channel linked from this repo.
